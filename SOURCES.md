# Sources

- Malaria data: user-provided PDF files in `data/raw/`.
- India district GeoJSON: `udit-001/india-maps-data`, downloaded from the repository CDN and saved as `data/raw/india_states.geojson`.
  - Repository: https://github.com/udit-001/india-maps-data
- Population denominators: Census 2011 state/UT population values, saved as `data/raw/state_population_2011.csv`.
  - Census geographic and administrative context: https://censusindia.gov.in/census.website/en/data/geographic
